CAHIER DES CHARGES COMPLET
🏍️ PROJET : SYSTÈME DE GESTION DES TAXIMOTOS

Nom du projet : Alika Mobility / Alika Gestion Terrain

1. 🎯 CONTEXTE ET OBJECTIF
📍 Contexte

Dans la ville de Bunia, les associations de taximotos fonctionnent de manière informelle avec :

Enregistrement manuel des membres
Paiement journalier en cash
Absence de traçabilité
Risques de fraude
Mauvaise gestion des données
🎯 Objectif du projet

Créer une application mobile simple et robuste permettant :

Digitaliser les membres
Suivre les paiements journaliers
Sécuriser les recettes
Améliorer la gestion des associations
2. 👥 UTILISATEURS CIBLES
🎯 Profils
1. Administrateur (Association)
Gère toute la plateforme
Accède aux statistiques globales
2. Chef de parking
Supervise un groupe de taximen

3. Recouvreur (terrain)
Enregistre les paiements
Utilise l’application sur téléphone
4. Taximan
Possède une carte (QR code / ID)
Paye quotidiennement
